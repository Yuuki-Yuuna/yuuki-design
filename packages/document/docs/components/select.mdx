## API

<font size='4'>Select</font>

继承了 [ArcoSelect](https://arco.design/react/components/select#api) 的所有属性，重写以下属性

| 参数名         | 描述                                            | 类型                                            | 默认值            |
| -------------- | ----------------------------------------------- | ----------------------------------------------- | ----------------- |
| value          | 选择器的值（移除了 LabeledValue）               | string \| string[] \| number \| number[]        | -                 |
| defaultValue   | 选择框的默认值（移除了 LabeledValue）           | string \| string[] \| number \| number[]        | -                 |
| request        | 用于请求选项                                    | (keyword?: string) => Promise\<LabeledValue[]\> | -                 |
| requestTrigger | 组件请求的时机                                  | RequestTriggerType[]                            | ['mount','input'] |
| resetOnBlur    | 是否在失去焦点时清除 option（必须开启聚焦请求） | boolean                                         | -                 |
| debounceWait   | 请求防抖的时间（ms）                            | number                                          | 800               |

<font size='4'>Select Reference Type</font>

| 参数名     | 描述         | 类型                 | 默认值 |
| ---------- | ------------ | -------------------- | ------ |
| getOptions | 获得所有选项 | () => LabeledValue[] | -      |
