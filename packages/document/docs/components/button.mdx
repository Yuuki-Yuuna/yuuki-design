## API

继承了 [ArcoButton](https://arco.design/react/components/button#api) 的所有属性，重写以下属性

| 参数名  | 描述           | 类型                                                                                      | 默认值    |
| ------- | -------------- | ----------------------------------------------------------------------------------------- | --------- |
| type    | 按钮类型       | 'default' \| 'primary' \| 'secondary' \| 'dashed' \| 'outline' \| 'text' \| 'text-normal' | 'default' |
| onClick | 点击按钮的回调 | (e: React.MouseEvent) => void \| Promise\<void\>                                          | -         |
