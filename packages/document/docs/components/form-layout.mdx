## API

<font size='4'>FormLayout</font>

| 参数名     | 描述                                    | 类型                                   | 默认值           |
| ---------- | --------------------------------------- | -------------------------------------- | ---------------- |
| className  | 节点类名                                | string \| string[]                     | -                |
| style      | 节点样式                                | React.CSSProperties                    | -                |
| layout     | FormItem 采用布局                       | 'horizontal' \| 'vertical' \| 'inline' | 'horizontal'     |
| labelCol   | label 的格栅 col 参数                   | ColProps                               | \{ flex: 'none' \} |
| wrapperCol | 内容的格栅 col 参数                     | ColProps                               | \{ flex: 'auto' \} |
| colon      | 子元素中 FormItem 的 label 是否带有冒号 | boolean                                | true             |
| asterisk   | 必填星号                                | boolean                                | true             |
| labelAlign | label 的对齐方式                        | 'left' \| 'right'                      | 'right'          |
