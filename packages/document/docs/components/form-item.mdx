## API

继承了 [ArcoFormItem](https://arco.design/react/components/form#api) 的所有**与表单校验无关**的属性，新增以下属性

<font size='4'>FormItem</font>

| 参数名   | 描述           | 类型    | 默认值 |
| -------- | -------------- | ------- | ------ |
| gridSpan | ⽹格布局占宽   | number  | -      |
| suffix   | 是否是后缀元素 | boolean | -      |
| asterisk | 必填星号       | boolean | true   |
